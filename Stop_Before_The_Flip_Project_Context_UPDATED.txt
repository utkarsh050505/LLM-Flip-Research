Research Goal

Working title:

Stop Before the Flip: Predicting Harmful Overthinking in Large Reasoning
Models

Core research question:

Can we predict, from the current reasoning prefix, whether continuing to
reason will turn a currently correct answer into an incorrect one, while
still allowing productive self-correction?

This is NOT a generic early stopping project.

The prediction target is the future probability of a negative flip
(Correct → Incorrect).

The project aims to build:

1.  Prefix-level dataset
2.  Hazard predictor
3.  Stopping controller
4.  Cross-model evaluation
5.  ACL/EMNLP-quality research codebase

------------------------------------------------------------------------

Research Roadmap

Phase 1 - Run a local reasoning model. - Save reasoning traces.

Phase 2 - Extract: - logits - hidden states - attentions - token
probabilities

Phase 3 - Detect: - answer changes - positive flips - negative flips

Phase 4 - Build prefix-level dataset.

Phase 5 - Train hazard predictor.

Phase 6 - Evaluate stopping controller.
